# Attyx v0.2.15

## Improvements

- Copy mode now works beyond the visible area — you can scroll and select text outside the current viewport. (#50)
- TUI apps like atuin, btop, and other tools using crossterm/ratatui now display your configured theme colors correctly instead of falling back to default terminal colors. (#43)
- Apps can now query the terminal's foreground, background, cursor, and palette colors (OSC 10/11/12/4), so color-aware tools get accurate responses matching your active theme. (#43)
- The `COLORTERM=truecolor` environment variable is now set automatically for better app compatibility. (#43)

## Bug Fixes

- Fixed pasting into popups (e.g. lazygit, search) — paste content is now routed correctly to the focused popup. (#49)
- Fixed fallback font sizing — symbols, braille patterns, and other characters rendered by fallback fonts are now correctly sized and aligned, most noticeably in tools like btop. (#46)
- Fixed high CPU usage on Linux (50–90% idle down to ~3%) by switching to an event-driven main loop that sleeps when idle and wakes instantly on new output. (#47, #42)
- Fixed Linux startup showing a blank screen with a blinking cursor until a key was pressed — the initial shell output is now processed and displayed immediately. (#48)
