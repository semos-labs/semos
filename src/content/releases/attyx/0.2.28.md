# Attyx v0.2.28

## Fixes

- **Images now work in session mode** — apps that use the kitty graphics protocol (like `timg`, `viu`, or `ranger`) previously failed to display images when running through the session daemon. The daemon now properly responds to kitty graphics queries, so image rendering works the same whether you're in a direct window or a session.
