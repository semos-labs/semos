# Attyx v0.3.14

## Better link clicks, split panes, and agent tabs

This release fixes a handful of everyday interactions that broke inside TUIs and split layouts.

### What changed

- **Modifier-click on links now works inside TUIs** — Cmd-click (macOS) and Ctrl-click (Linux/Windows) on OSC 8 hyperlinks and detected URLs open reliably, even when the running app captures mouse events. Hover feedback (hand cursor, underline) also shows up correctly when mouse motion reporting is active (#213)
- **Split panes keep your configured shell** — splitting a pane no longer falls back to a bare shell path, so your configured default program and PATH setup are preserved across splits (#212)
- **Agent tabs respond to clicks again** — clicking tabs or other split panes while an agent has mouse tracking active now focuses the right pane instead of being swallowed by the agent overlay (#210)
- **Per-agent status indicators** — tabs running OpenCode or Claude now show live status indicators, with room to grow support for more agents (#210)
