# Attyx v0.2.17

## Improvements

- **Smaller app bundle** — removed the bundled Sparkle framework. Attyx now handles updates with its own lightweight updater that uses macOS codesign verification, resulting in a noticeably smaller download and install size.
- **Faster CI builds** — the release pipeline no longer signs, bundles, or uploads Sparkle-related artifacts, making builds leaner and quicker.

**Full Changelog**: https://github.com/semos-labs/attyx/compare/v0.2.11...v0.2.17
