# Attyx v0.2.6

## Unicode in Tab Titles

Tab titles now correctly handle CJK/wide characters, combining marks, and special symbols. Previously, characters like `✻` or Chinese/Japanese/Korean text could appear as blank or garbled in tabs and the statusbar.

This also fixes a bug where OSC title sequences containing certain UTF-8 characters (e.g. `✻` U+273B) were silently truncated, causing tools like Claude Code to show broken or missing tab titles.

## Numpad Navigation (Linux) https://github.com/semos-labs/attyx/pull/31

Numpad keys now respect NumLock state on Linux. When NumLock is off, the numpad works as navigation keys (arrows, Home, End, Page Up/Down, Insert, Delete) — matching standard PC keyboard behavior. This also works correctly with keybinds and all input modes.

## Separate Dev and Prod Daemons

Development builds (`zig build`) and release builds now use separate daemon sockets and state files. You can run both side by side without them interfering with each other — no more killing the prod daemon to test a dev build.
