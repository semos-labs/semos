# Attyx v0.2.35

## macOS Permissions Done Right

Attyx now declares and signs with proper macOS entitlements, so apps running inside the terminal can request access to the camera, contacts, calendars, location, photos, and AppleScript automation — not just the microphone. When a CLI tool needs one of these, macOS will show its standard permission prompt instead of silently denying access.
