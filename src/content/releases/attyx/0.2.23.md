# Attyx v0.2.23

## Fixes

- **TUI apps render correctly after session switch** — live-updating terminal apps (htop, btop, dashboards, etc.) would sometimes show missing labels, headers, and static content when switching back to their session. Attyx now forces a full repaint so apps redraw cleanly after every session switch.

- **`kill-daemon` command works again** — the `attyx kill-daemon` subcommand was looking for the daemon socket in the old config directory instead of the XDG state directory, so it could never find a running daemon. Now correctly locates and kills the daemon process.
