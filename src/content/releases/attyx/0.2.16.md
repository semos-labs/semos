## What's Changed
# Attyx v0.2.16

## New Features

- **20 built-in themes** — Dracula, Gruvbox, Nord, Solarized, Tokyo Night, One Dark, Monokai, Rose Pine, Catppuccin Latte, Kanagawa, Everforest, GitHub Dark, Snazzy, Material, Iceberg, Palenight, and more — with dark/light/storm variants where applicable.
- **Theme picker** — open via command palette and pick a theme. Themes preview live as you navigate; confirm saves to your config, cancel reverts to the previous theme.
- **Themed UI everywhere** — overlays, statusbar, and tabs all follow the active theme's colors.
- **Full Kitty keyboard protocol support** — Shift+Enter, Alt+Enter, and Alt+Arrow word movement work correctly in apps that use the Kitty protocol.
- **Font size controls** — increase and decrease font size on the fly.
- **Native macOS menu items** for common actions.

## Improvements

- Session restoration now reconnects to the last active session instead of picking the last one in the list — you'll be right where you left off after a restart.
- Improved fuzzy search scoring in the session picker.
- Input fields now support Ctrl+W and Ctrl+U for word-jump and line-delete shortcuts.
- Improved updater with a more reliable publish workflow.

## Bug Fixes

- Fixed typing latency on Linux — disabled vsync and switched to manual ~60fps frame pacing so input is never blocked by swap buffers.
- Fixed font sizing and cell sizing on macOS.
- Fixed statusbar padding on macOS.
