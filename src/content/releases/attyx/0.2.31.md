# Attyx v0.2.31

## Capture Output with `--wait`

You can now capture the stdout of a tab or split when it exits using the new `--wait` flag. This is useful for scripting and automation — launch a command in a pane, and get its output back when it finishes.

```
output=$(attyx split vertical --wait --cmd 'echo hello')
```

The calling process blocks until the pane exits, then receives whatever the command printed to stdout.

## Pane-Aware Context Menu

Right-clicking now shows a richer context menu with pane management actions:

- **Split Vertical / Split Horizontal** — split the pane under the cursor
- **Rotate Panes** — cycle pane layout
- **Close Pane** — close the specific pane you clicked on

All actions target the pane under your mouse, not just the focused one. On macOS, "Open Config…" has also been added to the app menu for quick access to `attyx.toml`.

## Live Keybinds in the Command Palette

The command palette now shows your actual, currently bound shortcuts for each command — not hardcoded defaults. If you've customized or unbound a key, the palette reflects that. Hotkeys are formatted with platform-native symbols: `⌘⇧K` on macOS, `Ctrl+Shift+K` on Linux.
