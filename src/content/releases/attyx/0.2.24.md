# Attyx v0.2.24

## Fixes

- **No more crashes during daemon upgrade** — swapping to a new daemon version could trigger a crash on macOS (SIGTRAP in `_notify_fork_child`) because `fork()` is unsafe from background threads in multithreaded processes. All subprocess spawning now uses `posix_spawn`, which avoids the problem entirely. This also fixes the same class of crash when opening the config file or running detached commands from the popup.
