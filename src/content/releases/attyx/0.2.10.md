# Attyx v0.2.10

## Smoother Tab Switching in Session Mode

Switching tabs no longer causes a brief blank flash. Previously, the terminal engine was reset the moment you switched tabs — wiping the screen before the daemon had a chance to replay the session content. You'd see an empty terminal for a split second before everything reappeared.

Now the old content stays visible until replay data actually arrives, so tab switches feel instant and seamless. This also fixes apps running in the alternate screen (like vim or htop) that would briefly disappear during a tab switch.

## Safer Engine Reinitialization

The engine reinit that happens on tab focus now creates the new engine before tearing down the old one. If the new engine fails to allocate, the old one stays intact instead of leaving the pane in a broken state.
