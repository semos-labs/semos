# Attyx v0.3.11

## Xyron & Keyboard Fixes

A small patch release fixing Xyron connection reliability with multiple panes and a keyboard input issue with the Kitty protocol.

### What changed

- **Xyron handshake is now per-pane** — each pane independently establishes and manages its own Xyron IPC connection instead of sharing a single global one. This means opening new splits or tabs won't interfere with an existing Xyron session, and pane cleanup properly tears down its own connection (#200)
- **Key repeat works again with Kitty keyboard protocol** — when the Kitty protocol's `event_types` flag wasn't set, repeat key events were being silently dropped instead of treated as presses. Holding down a key now behaves as expected (#201)
