# Attyx v0.3.7

## Fixes

- **Cursor rendering** — the cursor trail effect no longer fires when the cursor was previously hidden, preventing ghost trails on reappear. Resize with reflow also keeps the cursor's screen-relative row intact, so the shell prompt doesn't duplicate or jump after a window resize (#188)
- **Stable tab titles** — renamed tabs now keep their label when you switch focus between panes. Previously, focusing a different pane inside a renamed tab would overwrite the title with the pane's process name (#186)
