# Attyx v0.4.13

## Agent-status dots now support Pi

Pi ([pi.dev](https://pi.dev)) joins Claude Code, Codex, and opencode as a recognized coding agent. Launch Pi in a pane and its per-tab status dot lights up while it's working and clears when it goes idle or exits — no setup required. Attyx drops a small status extension into Pi's auto-discovery directory the first time it sees `~/.pi`, and a shell-level backstop lights the dot on launch and clears it on exit even if the extension doesn't load. (#275)

One caveat: Pi has no approval/permission event, so there's no purple needs-input dot — same as Codex.
