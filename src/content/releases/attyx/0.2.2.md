## Persistent Sessions

Sessions now survive daemon restarts and crashes. Your workspaces are saved to disk automatically, so even if the daemon stops unexpectedly, you can pick up right where you left off.

## Recent Sessions

The session picker now shows recently closed sessions alongside active ones. Dead sessions appear dimmed so you can tell at a glance which are live and which are available to restore.

## Reorder Tabs

You can now rearrange tabs with keyboard shortcuts:

- **macOS:** `Cmd+Ctrl+Shift+Left/Right`
- **Linux:** `Ctrl+Alt+Shift+Left/Right`

## Bug Fixes

- Popups now correctly inherit the shell's working directory when launched from fullscreen apps like `vim` or `less`.
- Fixed visual artifacts when switching tabs in session mode.
