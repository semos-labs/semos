# Attyx v0.2.38

## Talk to Any Pane

**Pane targeting** — IPC commands like `send-keys`, `send-text`, and `get-text` now accept a `--pane`/`-p` flag so you can interact with any pane without switching focus. Target by pane index in the active tab (`-p 1`) or by tab and pane (`-p 2.0` for tab 2, pane 0). This makes multi-pane automation and agent workflows significantly smoother — no more focus-juggling to send input or read output from a background pane.
