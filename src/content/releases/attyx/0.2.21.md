# Attyx v0.2.21

## Improvements

- **Better text selection** — selection now survives scrolling. When you scroll while text is selected, the highlight tracks the same content instead of jumping or disappearing. Shift-click extends an existing selection on both macOS and Linux.

- **Open config with a shortcut** — press `Cmd+,` (macOS) or `Ctrl+,` (Linux) to open `attyx.toml` in your `$EDITOR`. If the config file doesn't exist yet, it's created automatically.

- **XDG state directory** — runtime files (session socket, daemon version, upgrade state, auth tokens, recent sessions) now live under `~/.local/state/attyx/` instead of `~/.config/attyx/`, following the XDG Base Directory spec. Existing files are migrated automatically on first launch — no manual steps needed.

## Fixes

- **Selection copy rewrite** — copy from mouse selection and copy mode now share the same scrollback-aware code path, fixing cases where copied text could be wrong or empty when the viewport was scrolled into history.

- **Selection not cleared on scroll** — scrolling no longer clears your selection. Only actions that produce input (typing, switching tabs, etc.) clear it.
