# Attyx v0.2.34

## Open Attyx from File Managers

You can now launch Attyx with a directory path as a positional argument — `attyx ~/Projects` — and it'll open with that as the working directory. This makes Attyx work seamlessly with file managers like Rox-Filer that launch terminals by passing a directory path.

## Microphone Access

Apps running inside Attyx that need microphone access (like Claude Code's voice features) will now trigger the macOS microphone permission prompt automatically. Previously, the permission was never requested, so voice-dependent features would silently fail.
