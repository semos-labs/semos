# Attyx v0.4.5

## A smarter session switcher

The session switcher now does more than list sessions — it shows you where your agents are and lets you spin up new workspaces without leaving the picker.

- **Agent status for every session** — each row gets a `ready / working / attention` badge, mirroring the status dots already on your tabs: green for ready, orange for working, purple for needs-input. All three counts are always shown, dimmed when zero, so you can tell at a glance which sessions are busy and which are blocked on you — even ones you don't have open. The badge is led by a robot glyph you can customize with the new `[sessions]` setting `icon_agent`. (#248)

- **Most-recent sessions first** — the list is now ordered by when you last opened each session, most recent at the top, instead of creation order. The sessions you actually switch between stay within easy reach. (#249)

- **Create new session folders from the picker** — type a name that doesn't match anything and a `Create "<name>"` row appears. Select it to create a new folder under your sessions root and jump straight into a session there. Names are checked for cross-platform safety so you won't end up with a folder you can't use. (#249)
