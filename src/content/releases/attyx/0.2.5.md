# Attyx v0.2.5

## Font Ligatures

Programming ligatures are now supported out of the box. Fonts like Fira Code, JetBrains Mono, and Cascadia Code will automatically render operator sequences (`=>`, `->`, `!=`, etc.) as connected glyphs.

Ligatures are enabled by default. To disable them, add to your config:

```toml
[font]
ligatures = false
```

You can also toggle via CLI with `--font-ligatures` / `--no-font-ligatures`. The setting is hot-reloadable.

## Bold and Italic Styles

Text styled with bold, italic, or bold-italic now renders correctly using the appropriate font variants. Previously these styles were ignored.

## Reflow and Resize in Session Mode

Terminal content now properly reflows when you resize the window in session mode. Previously, resizing could leave text garbled or misaligned when connected to a daemon-backed session.

## Popup Focus Lock

Popups now block system keybinds (tab switching, split navigation, etc.) while they're open. Previously, pressing a shortcut like Ctrl+Tab while a popup was open would switch tabs in the background, leaving the popup orphaned with no way to close it.

## Bug Fixes

- Fixed split pane mouse resizing — dragging separators now works reliably.
- Fixed working directory tracking when opening new tabs and popups.
