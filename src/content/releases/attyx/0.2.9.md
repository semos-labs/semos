# Attyx v0.2.9

## Instant Tab Switching

Switching tabs in session mode is now seamless. Previously, you could sometimes see raw escape sequences flash on screen after switching tabs — garbled text like `[?2026;2$y[?62c...` appearing below the prompt. This happened because terminal capability responses had to round-trip through the daemon, arriving too late for the shell to consume them properly.

The daemon now responds to terminal queries (DA1, DECRPM, kitty keyboard, etc.) directly at the PTY level, with zero round-trip latency. Tabs switch cleanly with no visual artifacts.

## Multi-Window Layout Sync

Tab and split changes now sync across multiple windows attached to the same session. Open a new tab or create a split in one window, and other windows on the same session update automatically to reflect the new layout.

## Better macOS Window Spawning

New windows opened from Attyx now group under the same dock icon and support Cmd+` switching between them, matching standard macOS app behavior.

## Statusbar Stability

The statusbar no longer briefly flashes empty or stale data when switching tabs or sessions. CWD and git widgets update immediately on the first frame after a switch.
