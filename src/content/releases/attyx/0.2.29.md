# Attyx v0.2.29

## IPC — Control Attyx from the Command Line

You can now control a running Attyx instance directly from your shell. The `attyx` binary doubles as a client — run commands like `attyx tab create`, `attyx focus right`, or `attyx send-text "hello"` and they're executed in the target instance over a local Unix socket.

This opens up scripting, keybind integration with external tools, and custom workflows without needing to touch Attyx's own config.

Available command groups:

- **`attyx tab`** — create, close, switch, move, and rename tabs
- **`attyx split`** — create vertical/horizontal splits, close, rotate, zoom
- **`attyx focus`** — move focus between panes (up/down/left/right)
- **`attyx session`** — list, create, kill, switch, and rename sessions
- **`attyx send-keys` / `send-text`** — send keystrokes or literal text to the active pane
- **`attyx get-text`** — read text from the active pane
- **`attyx list`** — show tabs, panes, and sessions (supports `--json`)
- **`attyx reload`** — reload config from disk
- **`attyx theme <name>`** — switch themes on the fly
- **`attyx popup`** — open a popup terminal overlay
- **`attyx run <cmd>`** — shorthand for `tab create --cmd <cmd>`

Use `--target <pid>` to aim at a specific instance, or let it auto-discover. Every subcommand supports `--help`.

## macOS Native Tabs Revamp

The native tab bar (`tab_appearance = "native"`) has been redesigned and now supports mouse-driven workflows:

- **Drag to reorder** — click and drag tabs to rearrange them. Other tabs shift to show the drop position.
- **Drag to tear off** — drag a tab far enough above or below the bar and it detaches into a new window.
- **Session dropdown** — when sessions are enabled, a new button next to "+" lets you switch sessions or create a new one from a dropdown menu.
- **Visual refresh** — flat, full-width tabs matching macOS Terminal.app style with proper separators, hover states, close buttons, and SF Symbol icons.
