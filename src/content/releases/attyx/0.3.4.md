# Attyx v0.3.4

> **Important:** The auto-updater in v0.3.3 was broken on Windows — it would download the update but fail to actually apply it. If you're on v0.3.3 or earlier, you'll need to manually re-download the latest binary from [GitHub Releases](https://github.com/nicholasrq/attyx/releases).

## Windows Updater Fix

Fixed a bug where the updater was looking for the wrong staging filename, causing updates to silently fail. The updater now correctly matches the build type and provides clear feedback on whether the update succeeded or failed.
