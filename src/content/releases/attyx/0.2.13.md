# Attyx v0.2.13

## Bug Fixes

- Fixed `attyx kill-daemon` leaving the UI frozen — the terminal appeared locked with no visible prompt until restarted. The session now recovers correctly and shell output flows through immediately.
- New sessions created after daemon reconnect now open in your last working directory instead of `/tmp`.
