# Attyx v0.2.37

## Claude Code Meets Your Terminal

**Claude Code skill** — Attyx now ships with a built-in Claude Code skill that lets Claude control your terminal via IPC. Split panes, send keystrokes, read screen output, manage focus — all from natural language. Install it with `attyx skill install` and Claude Code picks it up automatically. Use `/attyx` inside Claude Code to get started.

**Prettier CLI help** — The `attyx --help` output got a visual refresh with color-coded sections and cleaner formatting, so it's actually pleasant to scan. Falls back to plain text gracefully when piped or redirected.
