# Attyx v0.4.7

## Reliable agent status dots

This release makes the agent state indicators on your tabs dependable again — they no longer disappear or fail to appear when you're running agents across multiple synced panes.

- **Dots survive session switches** — switching between sessions used to clear the agent status dots from your tabs. They now stay put, so you can always tell at a glance which agents are working, waiting, or idle. (#255)

- **Dots show up on first launch** — if an agent reported its status the moment it started, before the view had finished focusing, the dot could be dropped and never appear. The status is now reshipped as soon as a pane comes into focus, so the first launch is as reliable as every one after it. (#255)
