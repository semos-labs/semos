# Attyx v0.2.40

## Smarter Session Creation

**Start sessions in any directory** — `attyx session create ~/Projects/myapp` opens a new session rooted in that directory and automatically names it after the folder ("myapp"). No more creating a session and then `cd`-ing into the right place.

**Background sessions** — By default, creating a session now switches to it immediately. If you'd rather spin one up without leaving your current session, pass `-b` or `--background`. Handy for pre-staging sessions you'll switch to later.

## Cross-Session Commands

**Target any session from anywhere** — The new `-s`/`--session <id>` flag lets you send IPC commands to a specific daemon session without switching to it first. Create tabs, send text, manage panes — all from whichever session you're currently in.

```sh
attyx session create ~/Projects/myapp              # create + switch, named "myapp"
attyx session create ~/Projects/myapp -b "build"   # background, custom name
attyx -s 3 tab create                              # create a tab in session 3
attyx -s 3 send-text "make test" -p 2              # send to pane 2 in session 3
```
