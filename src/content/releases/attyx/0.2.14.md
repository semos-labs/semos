# Attyx v0.2.14

## Bug Fixes

- Fixed working directory (CWD) being lost for tabs running TUI applications (e.g. Claude Code, vim, htop) in session mode. Previously, switching away from a TUI tab and back would clear the stored CWD, causing the statusbar widget to go blank and popups (like lazygit) to launch in the wrong directory. The daemon now remembers each pane's last reported CWD and shell PATH, and restores them when you switch back.
