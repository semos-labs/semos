# Attyx v0.2.26

## Improvements

- **Mouse support in popups** — popup terminals now handle mouse clicks, drags, scrolling, and movement. Apps like `fzf`, `lazygit`, or anything that uses mouse tracking will work properly inside popups.

- **Unified scrollback** — the terminal engine now uses a single ring buffer instead of separate grid and scrollback structures. Scrolling is faster, resizing reflows all content at once (no more orphan rows after repeated resizes), and session restore no longer flashes a `%` artifact or wrong-width prompt.

## Fixes

- **Viewport sync during full-screen apps** — fixed a timing issue where the viewport offset could get overwritten when a full-screen app scrolled, causing visual glitches.

- **Tab titles update reliably** — title changes in background tabs are now detected and reflected in the tab bar and status bar immediately, not just when you switch to that tab.

- **No more crash on session switch** — switching sessions while a tab was still initializing could crash; the event loop now guards against empty tab state.
