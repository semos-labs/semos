# Attyx v0.3.13

## Multi-Pane Overlay Fixes

This release fixes search highlights, completions, and overlays rendering in the wrong position when using split panes.

### What changed

- **Search highlights and overlays now align correctly in split-pane layouts** — previously, search result highlights and completion overlays were positioned relative to the pane's own grid rather than the full window, causing them to appear offset in multi-pane setups. They now render at the correct window coordinates across macOS, Linux, and Windows (#207)
