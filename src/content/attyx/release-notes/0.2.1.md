## Status Bar

Attyx now has a configurable status bar that sits at the top or bottom of your terminal window. It displays useful context at a glance — your current working directory, git branch and status, and the time — without leaving the terminal.

The status bar also serves as a tab strip, showing all open tabs with clickable titles for quick switching.

Widgets refresh on a configurable interval and react to directory changes automatically. You can add your own custom widgets by placing executable scripts in `~/.config/attyx/statusbar/`.

```toml
[statusbar]
enabled = true
position = "bottom"

[statusbar.cwd]
side = "left"
truncate = 2

[statusbar.git]
side = "left"

[statusbar.time]
side = "right"
24h = true
```

## Sessions

Sessions bring tmux-style workspace management to Attyx. Each session is a self-contained workspace with its own tabs, panes, and layout — managed by a lightweight background daemon.

- **Create a session** with `Ctrl+Shift+N`. It picks up the name from your current directory.
- **Switch sessions** with `Ctrl+Shift+S` to open the session picker. Filter by typing, navigate with arrow keys, and press Enter to switch.
- **Rename** a session with `Ctrl+R` in the picker.
- **Kill** a session with `Ctrl+X` in the picker (with confirmation).

Your tab and pane layouts are preserved when switching between sessions, so you can jump back to exactly where you left off.

## Native macOS Tabs

On macOS, you can now use native title bar tabs instead of the built-in overlay tab bar. Native tabs integrate with the system window chrome and follow your macOS appearance settings (dark/light mode).

Enable them in your config:

```toml
[tabs]
appearance = "native"
```

All tab keybindings work the same regardless of which style you use — `Cmd+T` for new tab, `Cmd+W` to close, `Cmd+1`–`9` for direct selection, and `Ctrl+Tab` / `Ctrl+Shift+Tab` to cycle.

## Command Palette

Press `Cmd+Shift+P` (macOS) or `Ctrl+Shift+P` (Linux) to open the command palette — a searchable list of every available action in Attyx.

Start typing to filter commands by name or description. Each entry shows its keyboard shortcut so you can learn the bindings as you go. Press Enter to execute, Escape to dismiss.

## Visual Mode

Visual mode gives you keyboard-driven text selection, inspired by Vim.

Enter copy mode with `Ctrl+Shift+Space`, then navigate with `h/j/k/l` or arrow keys. From there:

- `v` — character-wise selection
- `V` — line-wise selection
- `Ctrl+V` — block (rectangular) selection

Move the cursor to extend your selection, then press `y` to copy it to the clipboard.

You can also search within the terminal while in copy mode using `/` (forward) or `?` (backward). The search query appears in the status bar, and matches are highlighted as you type.

Numeric prefixes work too — `5j` moves down 5 lines, `3w` jumps 3 words forward.

Press `Escape` to step back through modes, or `y` to yank and exit.
