## Installation

```bash
bun add -g @semos-labs/aion
```

## Setup

Launch Aion for the first time:

```bash
aion
```

You'll be prompted to authenticate with your Google account. Aion will open a browser window for OAuth â€” grant calendar access and you're in.

## Usage

Once authenticated, Aion launches into the weekly view by default.

### Key Bindings

| Key | Action |
|-----|--------|
| `h` / `l` | Previous / next period |
| `t` | Jump to today |
| `n` | New event |
| `enter` | View event details |
| `d` | Delete event |
| `1` / `2` / `3` | Day / week / month view |
| `q` | Quit |
