---
title: LayoutRect
---
Computed layout rectangle for a node (returned by [useLayout](../functions/useLayout.md)).

`x`, `y`, `width`, `height` are the **outer** bounds (including border).
`innerX`, `innerY`, `innerWidth`, `innerHeight` are the **content** bounds
(after border and padding).

## Properties

| Property | Type |
| ------ | ------ |
| <a id="height"></a> `height` | `number` |
| <a id="innerheight"></a> `innerHeight` | `number` |
| <a id="innerwidth"></a> `innerWidth` | `number` |
| <a id="innerx"></a> `innerX` | `number` |
| <a id="innery"></a> `innerY` | `number` |
| <a id="width"></a> `width` | `number` |
| <a id="x"></a> `x` | `number` |
| <a id="y"></a> `y` | `number` |
