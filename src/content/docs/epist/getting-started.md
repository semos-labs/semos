## Installation

```bash
bun add -g @semos-labs/epist
```

## Setup

Launch Epist:

```bash
epist
```

On first launch, you'll authenticate with your Google account. Epist will open a browser window for OAuth â€” grant Gmail access and you're ready to go.

## Usage

Epist opens to your inbox by default.

### Key Bindings

| Key | Action |
|-----|--------|
| `j` / `k` | Next / previous email |
| `enter` | Open email |
| `c` | Compose new email |
| `r` | Reply |
| `a` | Archive |
| `d` | Delete |
| `/` | Search |
| `q` | Quit |
