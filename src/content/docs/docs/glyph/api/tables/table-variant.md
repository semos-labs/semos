---
title: 'TableVariant'
---

```ts
type TableVariant = "full" | "clean" | "clean-vertical";
```

Controls which grid lines are drawn.

| Variant | Surrounding border | Header separator | Row separators | Column separators |
|---|---|---|---|---|
| `"full"` | ✓ | ✓ | ✓ | ✓ |
| `"clean"` | ✗ | ✓ | ✗ | ✗ |
| `"clean-vertical"` | ✗ | ✓ | ✗ | ✓ |
