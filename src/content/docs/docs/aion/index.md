Aion is a full-featured calendar client that runs in your terminal. It supports **Google Calendar** and **CalDAV** (iCloud, Fastmail, Nextcloud, Radicale, and more) — view, create, and manage events without leaving the command line. Built with Glyph for a smooth, responsive interface.

## Features

- **Calendar views** — day, week, and month views
- **Event management** — create, edit, and delete events
- **Google Calendar sync** — native OAuth with PKCE, multi-account support
- **CalDAV support** — iCloud, Fastmail, Nextcloud, Radicale, and any CalDAV server
- **Keyboard-driven** — navigate everything without touching a mouse
- **Multiple calendars** — toggle visibility per calendar
