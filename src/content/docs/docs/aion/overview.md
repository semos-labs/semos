Aion is a full-featured Google Calendar client that runs in your terminal. View, create, and manage events without leaving the command line. Built with Glyph for a smooth, responsive interface.

## Features

- **Calendar views** — day, week, and month views
- **Event management** — create, edit, and delete events
- **Google Calendar sync** — real-time sync with your Google account
- **Keyboard-driven** — navigate everything without touching a mouse
- **Multiple calendars** — toggle visibility per calendar
