Epist is a terminal-native email client with **Gmail** and **IMAP/SMTP** support. Read, compose, and manage your inbox with keyboard-driven efficiency — use any email provider, or mix Gmail and IMAP accounts in a single client. Powered by Glyph for a clean, fast email experience right where you work.

## Features

- **Inbox management** — read, archive, delete, and label emails
- **Compose** — write and send emails with a built-in editor
- **Gmail + IMAP/SMTP** — Gmail OAuth and IMAP/SMTP for any provider, mix both freely
- **Search** — instant local filtering + remote server search
- **Labels & folders** — Gmail labels and IMAP folders with colored dots
- **Multiple accounts** — switch between Gmail and IMAP accounts
- **Keyboard-driven** — full workflow without leaving the terminal
