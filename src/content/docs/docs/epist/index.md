Epist is a terminal-native Gmail client. Read, compose, and manage your inbox with keyboard-driven efficiency. Powered by Glyph for a clean, fast email experience right where you work.

## Features

- **Inbox management** — read, archive, delete, and label emails
- **Compose** — write and send emails with a built-in editor
- **Search** — find emails with Gmail's powerful search syntax
- **Labels & filters** — organize mail just like in Gmail
- **Multiple accounts** — switch between Google accounts
- **Keyboard-driven** — full workflow without leaving the terminal
